# Awesome Tauri / Made with Tauri

## Source reference

- Site: https://madewithtauri.com/
- GitHub: https://github.com/tauri-apps/awesome-tauri

## Claim summary

Curated list of production Tauri apps. Notable shipping examples: Spacedrive (file manager, ~30k stars), Hoppscotch (API client, migrated from Electron), SilentKeys (privacy-first dictation with on-device ORT), Voxly (voice dictation), Watson.ai (meeting recorder). Also enterprise-adjacent: Ariadne (Git client), LumenTrack (crypto portfolio), LangForge (education).

## Relevance to our project

A3: Demonstrates a vibrant Tauri ecosystem. SilentKeys specifically uses on-device ORT — same combination as Image Browser (Tauri + Rust + ORT). The project sits inside an active production-app community.

## Specific takeaways

- Tauri is the *correct stack choice* for the on-device-ML category — multiple production apps use the exact same Rust+ORT+Tauri trio.
- Awesome-Tauri lists 100+ shipping apps. The category is mature, not nascent.
- The Hoppscotch migration (Electron→Tauri) is widely cited as a reference success story.

## Hype indicators

Moderate — community-curated lists naturally have some promotional voice, but the apps themselves are real.
