# Eagle — Commercial Image Organizer (macOS / Windows)

## Source reference

- Site: https://en.eagle.cool/

## Claim summary

Commercial photo / asset organizer. $29.95 one-time purchase. Local file organisation (no cloud), tags + folders + smart folders + colour filters. "Pinterest Visual Search" feature for similar-image lookup. Targeted at designers / content creators / artists.

## Relevance to our project

A3: A *commercial* comparable in the same category. Validates that "local-first image organizer" has a paying market. Eagle's success demonstrates the audience exists; Image Browser's open-source + Tauri stack is a defensible alternative position.

## Specific takeaways

- Eagle's "Pinterest Visual Search" positioning maps directly onto Image Browser's "View Similar" + "Pinterest-Style".
- Eagle uses a proprietary file format and licensing model; Image Browser is OSS + uses standard SQLite + ONNX.
- Comparable feature surface — gives the project a clear set of features that "professional users expect".

## Hype indicators

Moderate — commercial product, has marketing voice, but it's a real shipping app with a paying customer base.
