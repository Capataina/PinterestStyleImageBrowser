# EuroRust 2024 / 2025 — Rust + ML / Inference Track

## Source reference

- EuroRust 2024: https://rustfoundation.org/event/eurorust-2024/
- EuroRust 2025 talks: https://eurorust.eu/2025/talks/
- "Building a chatbot service" (EuroRust 2025): WGPU + WGSL + Tokio talk

## Claim summary

EuroRust has had ML / inference-adjacent talks across both 2024 and 2025. The community is actively producing content on Rust as an inference language. Talk volume is mid (3-5 ML-adjacent talks per year) — significant but not dominant.

## Relevance to our project

A1: The community signals around Rust+ML are real but not yet at "dominant track" level. This is consistent with the project sitting in a *growing but not yet mature* category — earlier-mover advantage.

A3: EuroRust includes Tauri-related talks; the desktop-app-in-Rust community has its own recognisable cohort.

## Specific takeaways

- The category is growing — 2025 has more ML/inference talks than 2024 had.
- A future portfolio action could include submitting a EuroRust talk on the project's choices (custom-WordPiece, ORT EP fallback, encoder-behind-trait).
